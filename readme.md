<h1>Sudoku Solver</h1>

<p>Node.js application</p>
<p>Enter sudoku pattern, e.g. '..9..5.1.85.4....2432......1...69.83.9.....6.62.71...9......1945....4.37.4.3..6..'</p>
<p>Sudoku is filled row by row, dots are empty spaces</p>
